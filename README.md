Dataset yang digunakan dalam tugas ini adalah dataset karyawan yang berisi informasi seperti usia, salary, pengalaman kerja, departemen, performance rating, serta status kerja (resign atau tidak).

Pada tugas ini, dataset digunakan untuk menganalisis faktor-faktor yang berkaitan dengan kinerja dan keberlangsungan kerja karyawan. Analisis pada tahap awal ini mencakup proses pemahaman konteks bisnis, tujuan analisis, kondisi awal dataset, serta rencana tahapan pengerjaan untuk proses analisis selanjutnya.
